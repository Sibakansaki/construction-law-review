## 功能

- 知識點卡片瀏覽，支援熟練／待加強狀態標記

- 考古題標注（100～114 年，含高考、普考、土木技師、司法三等等 6 種考試別）
- 篩選模式：全部 / 重點（曾考過或打星）/ 不常考
- AI 出題與申論答案批改
- AI 幫你想口訣
- 多選題複習
